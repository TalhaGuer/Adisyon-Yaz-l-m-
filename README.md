# Lezzet Büfe - Yazarkasa (Desktop Web)

Bu proje masaüstü tarayıcılarında çalışacak, geniş ekranlı bir yazarkasa/adisyon uygulamasıdır.

Başlamak için:

1. `src/firebase.js` içindeki `firebaseConfig` değerlerini kendi Firebase projenizle doldurun.
2. Bağımlılıkları yükleyin:

```bash
npm install
```

3. Geliştirme sunucusunu başlatın:

```bash
npm run dev
```

Notlar:
- Proje Vite + React + Tailwind ile yapılandırıldı.
- Termal fişler için `@media print` kuralları `src/index.css` içinde eklenmiştir.
